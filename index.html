Server Deployment

  • Note :-

    Use VPS SERVER for Deploy node js

    Recommended use ionos Server

  1. Open your terminal and connect to your VPS using SSH:
    2. ssh root@<your_vps_ip_address>
  2. Update the package list:
    3. sudo apt update
  3. Install Nginx:
    4. sudo apt install nginx -y
  4. Install curl:
    5. sudo apt install curl -y
  5. Install NVM (Node Version Manager):
    6. curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
  6. Reload the shell configuration:
    7. source ~/.bashrc
  7. Install Node.js using NVM:
    8. nvm install --lts
  8. Install pnpm package manager:
    9. npm install -g pnpm
  9. Install pm2 (Node.js process manager):
    10. sudo npm install -g pm2
  10. Clone projects to your server using Git in the /var/www/ directory.
  11. Install dependencies for both the frontend and server projects.
  12. Build your ReactJS project in the frontend directory:
    13. cd /var/www/<project-name>/frontend
    pnpm install
    pnpm build
  13. Build your NestJS project in the server directory:
    14. cd /var/www/<project-name>/server
    pnpm install
    pnpm build
  14. Run NestJS using pm2 in the server directory:
    15. pm2 start --name your-app pnpm -- start:prod
  15. Create an Nginx configuration file:
    16. sudo nano /etc/nginx/sites-available/your-app
  16. Adjust and add the following Nginx configuration:
    17. 
              server {
                  listen 80;
                  server_name your-domain.com www.your-domain.com;
              
                  location / {
                      root /var/www/<project-name>/frontend/build;
                      index index.html;
                      try_files $uri $uri/ /index.html;
                  }
              
                  location /server/ {
                      proxy_pass http://127.0.0.1:3000;  # Change the port if your NestJS server runs on a different port
                      proxy_http_version 1.1;
                      proxy_set_header Upgrade $http_upgrade;
                      proxy_set_header Connection 'upgrade';
                      proxy_set_header Host $host;
                      proxy_cache_bypass $http_upgrade;
                  }
              }
  17. Create a symbolic link to enable the Nginx configuration: 
    18. sudo ln -s /etc/nginx/sites-available/your-app /etc/nginx/sites-enabled
  18. Test the Nginx configuration:
    19. sudo nginx -t
  19. Restart Nginx to apply the changes:
    20. sudo systemctl restart nginx

Installation Steps

PNPM Package Management

  1. Check pnpm Installation: 
    pnpm --version
  2. If pnpm is not installed: 
    npm install -g pnpm

Server (Nest.js)

  1. Change Directory to Server: 
    cd server
  2. Setup .env file:

    Change .env.example file to .env

    Full field the .env file

  3. Install Dependencies and Start Server: 
    pnpm install
    pnpm start:dev

Doctor App Setup

  1. Open Project In Xcode
    • First make sure that you have installed the last version of X-code
    • Launch Xcode and navigate to the project folder. Open the .xcodeproj file to load your project into Xcode's workspace.
  2. Change App Name
    • Click on project navigator in general tab -> Disply Name -> (Enter New Name)
  3. Change App Identifier
    • Click on project navigator in signing & capabilities tab -> Signin -> (Enter New Bundle Identifier)
  4. Change BaseURL
    • Click on Model Folder -> Open APIManager.swift File -> Replace Your URL with baseUrl Parameter
  5. Change AppIcon
    • Navigate [Project_Directory] -> Open Assets -> Select AppIcon -> Replace Your New AppIcon
